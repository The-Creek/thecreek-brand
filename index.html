The Creek Brand Kit

The single source of truth for The Creek's brand assets, colors, typography, and voice. Designed to be used by staff, designers, and AI tools to produce consistently on-brand work.

Agent Instructions

Start with brand-kit.json — it contains all structured brand data in a single machine-readable file. For tone, logo usage, and writing rules, read brand-guidelines.md. All asset paths in the JSON are relative to the brand kit root. When producing branded output, default to Black (#231F20) and White (#FFFFFF) as the primary colors. Use Gotham Bold for headlines and CTAs, Bebas Neue Pro for subheads, and Avenir for body copy. Prefer the horizontal combination mark logo in most contexts. For ministry-specific content, apply the ministry's accent_color from the sub_brands array alongside black and white.

JSON Schema

Key Description
brand Legal name (Indian Creek Christian Church), brand name (The Creek), and website URL.
locations Primary campus address with lat/lng coordinates.
color_system.palette.primary Black (#231F20) — main brand color. Includes RGB, CMYK, and text_on value.
color_system.palette.secondary White (#FFFFFF) — co-primary. Used for logo reversal, backgrounds, and negative space.
color_system.palette.neutrals Dark Gray (#6D6E71, Pantone 424 C) and Medium Gray (#A7A9AC, Pantone 7543 C) with Pantone and CMYK values.
color_system.palette.accents Four accent colors: Blue (#95B1C1), Burgundy (#984445), Teal (#598D7F), Peach (#F79A70). Each with Pantone and text_on values.
color_system.interface Five-step neutral scale (softest → strongest) with light and dark mode values for UI contexts.
color_system.semantic Success, warning, danger, and info states — forecolor and background for light and dark modes.
typography.primary Gotham Bold — logo wordmark, headlines, and CTAs. Self-hosted OTF.
typography.secondary Bebas Neue Pro — subheadings. Self-hosted TTF.
typography.body Avenir — body copy. Upper/lowercase only; never all caps. Self-hosted TTF.
typography.accent Portrait Script — decorative accent only. Never for readable copy. Never all caps.
logos.lockups Five lockup types (combination_mark_horizontal, combination_mark_stacked, logo_mark, secondary_horizontal, secondary_stacked). Each with dark and light variant PNG assets.
logos.rules Logo usage rules: black or white only, preferred lockup order, clearspace, no alterations.
sub_brands Seven ministry brands, each with a unique accent_color and logo assets (combination_mark and wordmark in dark/light). Includes logo usage rules for internal vs. external contexts.
voice Brand voice summary, tone descriptors (warm, welcoming, clear, authentic, hopeful, community-focused), and language to avoid.
rules Top-level rules array — the primary guardrails for AI agents and designers producing branded output.
